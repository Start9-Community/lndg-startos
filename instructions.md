# LNDg

## Documentation

- [LNDg upstream README](https://github.com/cryptosharks131/lndg#readme) — the upstream project README, covering the auto-rebalancer, fee management tools, and dashboard features.

## What you get on StartOS

- **A web dashboard** for analyzing your LND routing node — channel performance, forwards, failed HTLCs, and fee management — backed by LNDg's own sqlite database that augments LND's data.
- **An auto-rebalancer and other background jobs** that keep working while you're not logged in.
- **An LND connection wired for you.** LNDg reads LND's TLS cert and admin macaroon directly from the LND dependency mount; you don't paste anything.
- **A StartOS-managed admin account.** Username and password are generated by an action on demand rather than baked into the image.

## Getting set up

1. Install **LND** from the marketplace first — LNDg is a frontend for LND and requires it as a dependency.
2. After LNDg installs, a **critical task** appears on its service page prompting you to **Create Admin Credentials**. Run the action — it generates a 22-character password and reveals both the username (`lndg-admin`) and the password. Save the password somewhere safe; it will not be shown again until you reset it.
3. Start the service.
4. Open the **Web UI** interface from the **Dashboard** tab and log in with `lndg-admin` and the password you just saved.

## Using LNDg

### Web interface

LNDg's web UI is its only user-facing surface. After logging in you land on the node dashboard, with channel tables, forwarding analytics, the auto-rebalancer controls, and fee management tools available from the navigation. All day-to-day configuration — auto-rebalancer settings, fee policies, channel views — happens inside the web UI, not in StartOS.

### Actions

- **Reset Admin Credentials** — generates a new random password for the `lndg-admin` Django superuser and reveals it. Use this if you've lost the password or want to rotate it. The next time the service starts, LNDg's Django superuser is re-synced to the new password automatically.
